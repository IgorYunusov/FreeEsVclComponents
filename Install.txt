Please see "Doc/ReleaseNotes_en.pdf"

0. Uninstall previous version!
1. Add "Source" folder to search path (Tools->Options->Delphi Options->Library->Library path)
2. Open "ErrorSoftGroup.groupproj" project group
3. Build "EsCore.bpl"
4. Build "EsVclCore.bpl"
5. Build "EsVclComponents.bpl"
6. Install "EsVclComponentsDesign.bpl"
7. Enjoy!

WARNING:
1)
Activate FixLoadPng in "Source/FreeEsVclComponents.inc" if you are having problems downloading PNG images in the Design time, 
because of some installed packages (ex: AlphaSkins)
